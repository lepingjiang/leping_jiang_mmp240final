RZA21505-OAE (Bitmap Font)

2019

Medium: Adobe Illustrator

A project for multimedia design class use Adobe Illustrator, that I need to make a decorative font with 1 by 1 squares, and blocks can’t be rotated in anyway, and the font needs to be readable.

The first problem is that a lot of bitmap font looks very similar, and I personally don’t want my design to be the same as the others.

I looked a lot of fonts and find there is a lot of fonts bit there isn’t a lot if font that is an out an outline, and I base my design on the goal of making an outline.
