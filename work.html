Candle fire with Arduino and p5.js

More Detail On This Project

2019

Medium: Html, CSS, p5.js, Arduino

A project for a multimedia programming class using p5.js and Arduino. The P5 sketch to respond to an input from a DHT humidity senor connected to the microcontroller. I wanted to make something related to fire so I chose the humidity sensor. I made a candle and fire graphic in Adobe Illustrator and used the values from the sensor to control the size of the fire and the number of flies on the screen.
